## 第四优先级：多路复用（比 tmux 更好）✅ 已完成

### 两种操作方式

| 操作方式 | 触发方法 |
|---------|---------|
| **右键菜单** | 在终端区域点击鼠标右键 |
| **快捷键** | 按对应的键盘组合键 |

### 和「桌面右键」是不是同一种界面？

**不是同一种东西，但流程可以类比。**

- 在 **Windows 桌面 / 文件夹空白处** 右键，弹出的是 **操作系统（资源管理器）画出来的原生菜单**：圆角、亚克力/云母、图标、子菜单，那是 **Explorer + WinUI/壳** 提供的，**不是** 终端程序自带的控件。
- 在 **WezTerm 终端区域** 右键，弹出的是 WezTerm 自己的 **InputSelector**：终端里叠一层 **文字列表**（上面那句英文提示 + 用 ↑↓ 选、回车确认）。**交互**上可以当成「先右键再选功能」，和桌面一样是分步操作；**长相**做不到和桌面右键 **100% 一样**，因为 WezTerm **目前不能在窗口里嵌入与桌面完全相同的 Win32 右键菜单**。这不是配置写错，是这类终端模拟器的普遍限制。
- 若你更看重 **和系统一模一样的菜单外观**：在 **真实桌面或文件夹里** 用右键（例如已提供的「在此处打开 WezTerm」注册表思路），或接受终端内为 **列表式菜单**；**功能**（分屏、工作区等）仍通过当前右键列表完成。

### 使用方法（右键菜单方式）

这就是你要的效果：**在终端里右键 → 出列表 → 选功能 → 有的会再问一句**（和下面步骤一致）。

1. **在终端画面里（有光标、能打字的那块区域）点一下右键并松开**（配置为**松开右键**时出菜单，**不必一直按着**右键再选；不要在 Windows 桌面空白处点——那是系统菜单，不是 WezTerm）。
2. 菜单标题会提示键盘操作。用 **↑↓** 移到要用的那一项，按 **Enter** 确认；想关掉菜单不操作时：按 **Esc**，或选最后一项 **「退出菜单」**。
3. **例如切换/创建工作区**：选 **「切换或创建工作区…」** → 回车后会出现输入框 → **输入工作区名字**（如 `project-a`）→ 再回车，即切换到该工作区（没有这个名字会自动建一个）。
4. **例如左右分屏**：选 **「左右分屏」** → 回车，当前窗格会立刻左右分屏（无需再输入）。
5. **保存并恢复（可选）**
   - 右键 → **保存当前布局**（或 `Ctrl+Shift+Alt+S`）
   - 需要关窗口时：右键 → **保存布局并关闭本窗口**
   - 下次打开 WezTerm 后，仍可用右键 → **切换或创建工作区…** 输入之前的工作区名，或 `Ctrl+Shift+M` 列出工作区再选

**若右键仍是旧菜单或没反应：** 确认 WezTerm 读的是当前这份配置（例如 `C:\Users\lenovo\.wezterm.lua` 与仓库里 `e:\WezTerm\.wezterm.lua` 已同步），然后 **完全退出 WezTerm 再打开**，或 `Ctrl+Shift+R` 重载配置。

### 使用方法（快捷键方式）

适合键盘党，按相应快捷键直接触发功能。

### 基于工作区的"类多路复用"原理

WezTerm 原生多路复用（mux）在 Windows 上不如 Linux/macOS 完善，我们使用**工作区+自动保存**来实现类似效果：

1. **创建专用工作区**：运行长任务前创建工作区（如 `training`）
2. **切换工作区**：不同工作区有独立的环境，互不影响
3. **保存/恢复**：工作区的目录和分屏状态会被保存
4. **多客户端**：多个 WezTerm 窗口可以打开同一名称的工作区（需手动切换）

## 第五优先级：视觉增强（Dracula + ROS/长时开发向）✅ 已实现

| 功能 | 说明 |
|------|------|
| **主题** | **Dracula-ROS** 默认：`#282a36` 深灰底（非纯黑），ANSI 红/黄/绿分层清晰，便于 **ROS 日志、rqt、长时间扫屏** |
| **状态栏** | 右侧保留**最小必要信息**：**工作区名 + 电量 + 时间**，弱化配色与分隔，减少视觉干扰 |
| **背景** | **纯色底**（已去掉多色纵向渐变，减少干扰彩色输出）；**Acrylic** + `window_background_opacity ≈ 0.93`（约 7% 透明感） |
| **字体** | **JetBrains Mono**，连字开启；`font_size = 14` |
| **标签栏** | **Fancy tab bar**；标签与 **Dracula** tab_bar 色一致 |
| **光标与滚动条** | `cursor_thickness = 2`；滚动条拇指 `#6272a4`（当前版本不支持单独设轨色）；**视觉铃** 不显式配置（避免配置错误弹窗；`audible_bell = Disabled`） |
| **配置重载** | `automatically_reload_config = false`：改配置后手动 `Ctrl+Shift+R`，避免编辑半成品时多窗口连弹错误框 |
| **WSL 启动目录** | 通过 `wsl_domains.default_cwd = '/home/lenovo'` 固定默认进入 home，避免继承到 `/mnt/c/Users/lenovo` |
| **按目录换色** | `/prod/`、`production` → **Dracula-Warn**；`/staging/`、`staging` → **Dracula-Staging**；规则在 `PATH_THEME_RULES`（可加 catkin 工作空间等子串） |

### 测试建议

1. 新开 WezTerm，确认默认落在 `~`（如 `/home/lenovo`），不再继承到 `/mnt/c/Users/lenovo`。
2. 看状态栏是否只显示 **工作区名 / 电量 / 时间**，且视觉不抢眼。
3. `cd` 到路径里带 `staging` / `production` 的目录（或改规则匹配你的目录名），观察整体配色是否仍按目录切换。
